# Credit Risk Analysis

## Project Overview
This project focuses on analyzing borrower and loan data to identify factors contributing to loan default risk using SQL and Microsoft Excel. The objective of this project is to perform data-driven credit risk assessment and generate business insights that support better lending decisions and risk management strategies.

---

## Tools & Technologies Used
- SQL
- Microsoft Excel
- Pivot Tables
- Data Visualization
- KPI Reporting

---

## Dataset Information
The dataset contains borrower and loan-related information including:
- Loan Amount
- Interest Rate
- Loan Grade
- Home Ownership
- Loan Intent
- Past Default History
- Age Group
- Default Status

---

## Project Objectives
- Analyze loan default patterns
- Identify high-risk borrower segments
- Perform loan grade and interest rate risk analysis
- Generate KPI-based dashboard reporting
- Support business decision-making using analytical insights

---

## Dashboard KPIs
- Total Loans
- Total Loan Amount

- Default Rate
- Average Loan Amount

---

## Key Insights
- Borrowers with previous defaults showed significantly higher default rates.
- Higher interest rate groups contributed to increased loan default probability.
- Loan grades C and D recorded the highest number of defaults.
- Renters demonstrated comparatively higher default rates than homeowners.
- Medical and education-related loans contributed heavily to default counts.
